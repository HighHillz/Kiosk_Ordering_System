# Kiosk Ordering System

This project evolved from a Grade 12 Python GUI (v1) to a containerized Full-Stack Web App (v2). It serves as a comprehensive demonstration of how a legacy desktop application can be modernized into a scalable web platform.

![Latest Version](https://img.shields.io/badge/Latest%20Version-2.0.0-blue.svg)
![License](https://img.shields.io/badge/License-MIT-black.svg)

## Project Structure

This repository is organized into two main versions:

### [Version 1: Legacy](./v1%20-%20Legacy/)
The initial desktop-based implementation built using Python and Tkinter. This version focuses on core functionality with a traditional GUI, serving as a "time capsule" of the original project.

### [Version 2: Modern Web](./v2%20-%20Modern_Web/)
The modern web-based implementation featuring a full-stack architecture with FastAPI and React. This version includes advanced features like a Kitchen Display System, dynamic branding, and multi-tenancy support, all containerized with Docker.
